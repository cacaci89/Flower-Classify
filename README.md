# Flower-Classify

## 使用情境說明
過年過節、婚喪喜慶、告白探病種種場合都會需要準備花束，人到了花店
或是路邊經過看到又怕問了店員或亂摘被人覺得被常識或亂送禮，或是在探索
大自然的你又不想透過顏色外觀一個個詢問 Google 這時就可以使用我們的花
卉辨識 App 來替你解決困擾。
可以用於 APP 裡面來辨識是哪種花，人在戶外，看到好看的花，但不知道
是什麼，打開 APP，就可以知道了。

## 用到的技術與工具
 * 遷移學習：Resnet50
 * 物件偵測：Yolov5
 * 網頁：Django、CSS
